# Backend Express/NodeJS

This project was created using Node v24.

## Technologies

This application server is using the following dependencies:

- Node v24 x nodemon
- TypeScript
- ExpressJS
- Jest (tests)
- eslint x prettier (linting)
- dotenv CLI (locally)

## Local Development

### Running

To start a local development server, run:

```bash
pnpm run dev
```

### Testing

To start the unit tests, run:

```bash
pnpm run test
```

To use watch mode, run:

```bash
pnpm run test:watch
```

### Linting

To launch linting checks, run

```bash
pnpm run lint
```

To fix it automatically, run:

```bash
pnpm run lint:fix
```
